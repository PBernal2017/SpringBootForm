# SpringBootForm
Proyecto Spring Boot utilizando
Thymeleaf, Spring DevTools y Spring Web.
Funciones:
- Formulario de registro.
- Muestra los resultados en otra vista.
- Implementa interceptores y validaciones.
- MVC
- Estilos con Bootstrap
